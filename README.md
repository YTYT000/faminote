# 概要
FamiNote（ファミノート）は大切な家族であるペットの健康管理アプリです。  
日々の記録を付けることで、  
- 介護  
- 通院  
- 治療  
- 自宅療養

など、ケアが必要な家族の生活をサポート出来ます。
